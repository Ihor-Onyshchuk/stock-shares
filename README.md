# Stock-Shares

This app uses [JSON API](https://iexcloud.io/docs/api/) that makes financial data and services accessible to everyone.

## Getting started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

IEX Cloud authenticates your API requests using your accountâ€™s API tokens. To use any IEX Cloud API, you must pass an API token with each request. If you do not include your API token when making an API request, or use one that is incorrect or disabled, IEX Cloud returns an error.

### Usage
* Clone project
* Run `npm install`
* Provide your own API key
* Run `npm start`

Runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
